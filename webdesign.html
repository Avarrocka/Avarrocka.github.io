RuneSeeker [Back To Top]

Website
RuneSeeker is a JSON parser of RuneScape's GE Item Database that offers users in depth information about items found in RuneScape
Not an official application nor otherwise affiliated with RuneScape or JaGeX.

RuneSeeker is a web-based application that searchs the RuneScape's Grand Exchange database using the Official RuneScape API to retrieve information about each item. It provides the user with a fast and efficient way to access otherwise obscure or uncommon items. By processing the users Query, RuneSeeker makes an API call to RuneScape's Grand Exchange API and recieves back a JSON string with all the relevant information. RuneSeeker parses this information and formats it into a neat information module for the user.

RuneSeeker tracks the properties of each item, what type of item it is (combat, furnishing, food, etc.), whether or not it is an exclusive item (members-only), and the monetary value of the item by different standards. It also provides financially savvy users with a graph to track market trends regarding the item, helping to illustrate patterns in the items monetary fluctuation.

In creating RuneSeeker, I learned about JSON, AJAX, BootStrap, and PHP.

IvyGlobal Diagnostics [Back To Top]

Website
IvyGlobal Diagnostics is a SAT/ACT prep tool used to test student progression with mock tests.
I do not claim ownership nor attempt to redistribute the services of this site.

IvyGlobal is a SAT/ACT preparation company that helps students in Canada and America prepare for the Standardized Aptitute Test, and the American College Testing Standardized Test. We designed IvyGlobal diagnostics to help tutors and students get a good idea of how they're progressing with their SAT/ACT studies.

Teachers can create test banks for Math, Reading - passage, Reading - article, and Writing subjects, as well as questions for secondary test topics such as Biology and Physics. These mock-tests are stored server-side.

In helping create IvyGlobal diagnostics, I learned to work with Javascript, AJAX, MathJax, LaTeX, and Microsoft SQL.

CollegeGrid ACT/SAT Database [Back To Top]

Website
CollegeGrid ACT/SAT Database is a information bank of US colleges and their related SAT information.
I do not claim ownership nor attempt to redistribute the services of this site.

CollegeGrid is a repository of information regarding American colleges and universities. Information such as average tuition, male to female ratio, and rate of undergraduate and masters degrees awarded per year can all be found on this site. CollegeGrid also has a system to predict a student's chances (MyChances) of being accepted to a specific college based on a holistic analysis of predicted SAT/ACT scores, placement in their class, and extracurriculars. CollegeGrid also recommends colleges to users, based uniquely on SAT/ACT scores.

Students are able to analyze their chances of getting into the colleges and universities that they desire, with their predicted or actual SAT/ACT scores. They are able to see the univerisites that are par/reach/safe for them, with their current scores, and plan accordingly. University information is updated yearly from official sources, so information is guarenteed to be accurate and relevant.

In helping create CollegeGrid, I learned to work with Javascript, AJAX, ASP, VBScript, ASP.net, and Microsoft SQL.

IvyGlobal Classroom [Back To Top]

Website
IvyGlobal Classroom is a virtual learning and course management environment.
I do not claim ownership nor attempt to redistribute the services of this site.

IvyGlobal Classroom is a tool that helps teachers manage their virtual classroom. Students can register for a class and see the syllabus for the class, detailed lesson plans regarding a class, as well as all classes for which they are registered. Teachers are able to monitor student performance, create tests, new classes, and assign homework/create lesson plans for students. This highly streamlined and efficient system creates an easy to use and intuitive virtual learning environment.

IvyGlobal classroom offers advanced analytics for each class, including test scores, individual student performances, and average performances. Teachers can use these helpful analytics to cater and adjust the direction of their course as they see fit. They are also able to post notes and reminders to students, for homework help or important reminders.

In helping create IvyGlobal Classroom, I learned to work with advanced CSS, AJAX, Angular JS, ASP.net, and MongoDB.
