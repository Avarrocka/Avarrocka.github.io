RuneSeeker [BETA]

About

I created RuneSeeker as a personal project to try to interface with the RuneScape Grand Exchange API. However, due to a slew of unforseen errors and bugs (such as GitHub pages not supporting ASP and PHP), as well as RuneScape API problems (not supporting Cross-origin resource sharing), it appears that the working demo has actually crashed and stopped working ☹

This project was originally intended to be like GW2Spidy, a site that has up to date information including pricing, descriptions, and other useful information about items.

This project may be revisited in the future, as fixes are thought of and more experience is earned.
